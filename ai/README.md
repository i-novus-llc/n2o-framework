N2O skill - это навык, которым вы можете дополнить используемый вами ИИ, 
чтобы он умел создавать и редактировать файлы написанные с использованием N2O Framework.

**Как использовать навык n2o-skill?** 

1. Добавить это навык. Для этого скопировать папку `n2o-skill` в папку с глобальным навыками ИИ или навыками для проекта. 
Для claude-code это будет папка `.claude/skills`. Для других ИИ проверьте что указано в документации ИИ.
2. В сессии с ИИ запустить навык командой `/n2o-skill` ИЛИ в настройках проекта указать, чтобы этот навык был активен для любой сессии.
Для claude-code это указание необходимо добавить в `CLAUDE.md` файл. 
3. Установить на компьютер python если он раннее не был у вас установлен. Необходимо для валидации xml файлов.
4. в запросах ИИ указать, что вы хотите создать\редактировать файл с использованием N2O Framework. 

**Что умеет n2o-skill?**

1. Создавать\редактировать любые компоненты из N2O Framework.
2. Исправлять ошибки связанные с xml конфигурациями N2O Framework.
3. Переводить xml файлы на последние версии xsd схем. 

_Замечание_
На текущий момент n2o-skill работает с последней версией N2O Framework.
