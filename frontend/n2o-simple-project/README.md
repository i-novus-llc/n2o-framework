# n2o-simple-project

`n2o-simple-project` — простое React-приложение на Vite для **локального дебага фронта** и быстрой проверки изменений в `n2o-framework`.

Проект входит в workspace `frontend` и подтягивает `n2o-framework` как зависимость из монорепозитория (удобно для разработки “вживую”).

## Использование

- быстро поднять UI и проверить, что `n2o-framework` собирается/работает;
- локально дебажить виджеты/страницы/инфраструктуру `n2o-framework`;
- посмотреть, как базово подключается `N2O` и фабричная конфигурация (`createFactoryConfig`);

## Быстрый старт (из `frontend/`)

* Установка зависимостей
* build
* compile:watch (если нужно следить за изменениями стилей или n2o-components)
* start:demo

## Структура

* `src/index.tsx` — вход и инициализация `N2O`
* `src/index.css` — локальные стили демо
