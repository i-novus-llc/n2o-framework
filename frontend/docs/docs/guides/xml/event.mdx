### &lt;on-change&gt; {#on-change}

Событие, срабатывающее при изменении модели данных

**Атрибуты**

| Наименование | Тип                                                        | Описание                                                         | Значение по умолчанию | Обязательность |
|--------------|------------------------------------------------------------|------------------------------------------------------------------|-----------------------|----------------|
| id           | Строка                                                     | Идентификатор события                                            |                       | !              |
| datasource   | Ссылка на [источник данных](../xml/datasource#datasources) | Источник данных, в котором будут отслеживаться изменения         |                       |                |
| model        | resolve edit filter multi datasource                       | Модель источника данных, в которой будут отслеживаться изменения | resolve               |                |
| field-id     | Строка                                                     | Поле источника данных, в котором будут отслеживаться изменения   | Вся модель            |                |
| action-id    | Ссылка на [действие страницы](../xml/page#actions)         | Действие страницы, которое будет вызвано при изменении данных    |                       |                |

**Тело**

[Список действий](../xml/action)

**Пример**

```xml
<on-change datasource="ds" field-id="name">
    <invoke>...</invoke>
    <alert>...</alert>
</on-change>
```

[Подробнее о событии on-change](../manual/events#on-change_event)



### &lt;stomp-event&gt; {#stomp-event}

Событие, приходящее через STOMP протокол

**Атрибуты**

| Наименование | Тип    | Описание              | Обязательность |
|--------------|--------|-----------------------|----------------|
| id           | Строка | Идентификатор события | !              |
| destination  | Строка | Место назначения      |                |

**Тело**

&lt;alert&gt; - [Действие уведомления](../xml/action#alert)

**Пример**

```xml
<stomp-event id="ev1" destination="/notifications">
    <alert>...</alert>
</stomp-event>
```

[Подробнее о событии stomp-event](../manual/events#stomp-event)
