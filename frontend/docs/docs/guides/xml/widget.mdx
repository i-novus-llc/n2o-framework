|master-field-id        |Ссылка на поле выборки                   |Ссылка на поле выборки в master виджете. Используется вместе с depends-on |id                                                                                     |

|master-param           |Строка                                   |Параметр фильтра                                                          |                                                                                       |

|class                  |Строка                                   |CSS класс, который будет добавлен к виджету                               |                                                                                       |
|style                  |CSS свойство                             |Стиль виджета                                                             |                                                                                       |
|name                   |Строка                                   |Наименование виджета                                                      |                                                                                       |
|icon                   |[CSS класс](http://fontawesome.io/icons/)|Иконка виджета                                                            |                                                                                       |
|query-id               |Ссылка на [выборку](../xml/query)        |Выборка виджета                                                           |                                                                                       |

|size                   |Число                                    |Количество записей, которое отображает виджет                             |10                                                                                     |

|auto-focus             |true false                               |Автоматическая установка фокуса на виджете                                |true                                                                                   |

#### &lt;actions&gt;

Действия над виджетом.

:::note

Действия не видимы сами по себе, но на них можно ссылаться, например, из [кнопок](#button_in_widget).

:::

**Атрибуты**

|Наименование|Тип                |Описание                      |Значение по умолчанию|
|------------|-------------------|------------------------------|---------------------|
|generate    |crud, submit, close|Генерация стандартных действий|                     |

**Пример**

```xml
<actions generate="crud">
  <action>...</action>
  <action>...</action>
  <action>...</action>
</actions>
```

##### &lt;action&gt;

Действие над виджетом.

**Атрибуты**

|Наименование|Тип                                      |Описание                                                 |Значение по умолчанию                                                                                           |
|------------|-----------------------------------------|---------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
|id          |Строка                                   |Идентификатор действия                                   |                                                                                                                |
|name        |Строка                                   |Наименование                                             |Вычисляется по операции объекта                                                                                 |
|icon        |[CSS класс](http://fontawesome.io/icons/)|Иконка действия                                          |Вычисляется, если `id` действия из списка стандартных действий.                                                 |
|default     |true false                               |Действие по умолчанию. Вызов по двойному клику или Enter.|Вычисляется, если `id` действия из списка стандартных действий.                                                 |
|hotkey      |Сочетание клавиш                         |Горячая клавиша вызова                                   |Вычисляется, если `id` действия из списка стандартных действий. Например, для `create` горячая клавиша `Insert`.|
|visible     |true false                               |Видимость действия                                       |true                                                                                                            |
|enabled     |true false                               |Доступность действия                                     |true                                                                                                            |

**Тело**

[Обработчик действия](../xml/action)

**Пример**

```xml
<action id="close">
  <close/>
</action>
```

#### &lt;toolbar&gt; {#toolbar_in_widget}

Меню управляющих кнопок виджета.

**Атрибуты**

|Наименование|Тип                                                                                     |Описание                    |Значение по умолчанию|
|------------|----------------------------------------------------------------------------------------|----------------------------|---------------------|
|place       |topLeft, topRight, bottomLeft, bottomRight, topCenter, bottomCenter                     |Место расположения меню     |topLeft              |
