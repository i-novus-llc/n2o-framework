#### &lt;button2&gt; {#button_in_button}

Кнопка.

**Атрибуты**

|Наименование             |Тип                                     |Описание                                                                                                                                                                                        |Значение по умолчанию|
|-------------------------|----------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------|
|action-id                |Ссылка на [действие страницы](#action)  |Действие страницы, которое будет вызвано при нажатии на кнопку                                                                                                                                  |                     |
|rounded                  |true false                              |Круглая форма кнопки                                                                                                                                                                            |false                |
|validate                 |true false                              |Валидирование форм при нажатии на кнопку.                                                                                                                                                       |true                 |
|validate-datasources     |Список идентификаторов [источников данных](../xml/page#datasource)|Источники данных, которые будут валидироваться                                                                                                                        |Источник данных текущего виджета|
|confirm                  |Строка                                  |Показать предупреждение перед выполнением действия.                                                                                                                                             |false                |
|confirm-type             |Строка                                  |popover modal                                                                                                                                                                                   |modal                |
|confirm-text             |Строка                                  |Текст предупреждения.                                                                                                                                                                           |                     |
|confirm-title            |Строка                                  |Заголовок окна предупреждения.                                                                                                                                                                  |                     |
|confirm-ok-label         |Строка                                  |Текст кнопки подтверждения.                                                                                                                                                                     |                     |
|confirm-cancel-label     |Строка                                  |Текст кнопки отмены.                                                                                                                                                                            |                     |
|model                    |resolve filter edit multi               |Модель, над которой выполнится действие. Если модель пустая, то кнопка будет заблокирована.                                                                                                     |resolve              |
|visible                  |Строка                                  |Видимость по умолчанию.                                                                                                                                                                         |                     |
|enabled                  |Строка                                  |Доступность по умолчанию.                                                                                                                                                                       |                     |
|datasource               |Идентификатор [источника данных](../xml/page#datasource)|Идентификатор источника данных, над которым будет выполнена операция                                                                                                            |Источник данных текущего виджета|
|disable-on-empty-model   |true false auto                         |Недоступность кнопки при пустой модели. `true` - всегда недоступна,`false` - всегда доступна, `auto` - недоступна только при пустой модели resolve, для всех остальных моделей доступна всегда. |auto                 |

**Тело**

[Обработчик действия](#Actions),
[Зависимости компонента](#_Зависимости_компонента)

**Пример**

```xml
<button action-id="create" enabled="{type=='type1'}"/>
```

```xml
<button
  label="Добавить в реестр"
  icon="fa-plus"
  type="icon">
  <open-page page-id="register" action-id="create"/>
</button>
```

##### &lt;sub-menu&gt; {#sub_menu_widget}

**Атрибуты**

|Наименование    |Тип       |Описание                        |Значение по умолчанию|
|----------------|----------|--------------------------------|---------------------|
|generate        |Строка    |Генерация стандартных кнопок.   |                     |
|show-toggle-icon|true false|Показывать иконку переключателя.|true                 |
|visible         |true false|Условие видимости кнопки.       |true                 |

**Пример**

```xml
<sub-menu>
  <menu-item>...</menu-item>
  <menu-item>...</menu-item>
  <menu-item>...</menu-item>
</sub-menu>
```

##### &lt;enabling&gt;

Условие доступности

**Атрибуты**

|Наименование |Тип                                                     |Описание                                          |Значение по умолчанию           |
|-------------|--------------------------------------------------------|--------------------------------------------------|--------------------------------|
|datasource   |Идентификатор [источника данных](../xml/page#datasource)|Идентификатор источника данных вычисления условия |Источник данных текущего виджета|
|model        |resolve, filter                                         |Модель виджета, по полю которого будет фильтрация |                                |
|message      |Строка                                                  |Сообщение о причине недоступности                 |                                |

**Тело**

Java Script выражение


**Пример**
```xml
<dependencies>
  <enabling datasource="main" model="resolve">type.id == 1</enabling>
  <!-- Поле доступно, если type.id равен 1 -->
</dependencies>
```

##### &lt;visibility&gt;

Условие видимости

**Атрибуты**

|Наименование |Тип                                                     |Описание                                                |Значение по умолчанию           |
|-------------|--------------------------------------------------------|--------------------------------------------------------|--------------------------------|
|datasource   |Идентификатор [источника данных](../xml/page#datasource)|Идентификатор источника данных вычисления условия       |Источник данных текущего виджета|
|model        |resolve, filter                                         |Модель виджета, по полю которого будет фильтрация       |                                |

**Тело**

Java Script выражение

**Пример**

```xml
<dependencies>
  <visibility datasource="main" model="resolve">type.id == 1</visibility>
  <!-- Поле видимо, если type.id равен 1 -->
</dependencies>
```
