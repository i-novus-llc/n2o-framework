### Изменения в XML API {#changes-xml-api}

- Добавлено новое действие `<alert>` для отображения сообщений. Поле `<alert>` и действие `<alert>` приведены к общему виду.
Вследствие чего из поля `<alert>` удалены атрибуты `fade`, `tag` и `footer`, атрибут `header` переименован в `title`. Текст для
отображения в `<alert>` теперь задается в новом атрибуте `text`.

### Изменения в сборке Frontend {#changes-frontend-build}

- Подняты версии используемых библиотек. Обновите их в своем проекте в package.json или поставьте `*`, чтобы использовать транзитивно из N2O
    "history": "4.9.0"
    "redux": "4.1.2"
    "redux-actions": "2.6.5"
    "redux-form": "8.3.8"
    "react-router-dom": "5.2.0"
    "react-redux": "7.2.6"
    "redux-saga": "1.1.3"

- Отказ от использования библиотеки "redux-batched-actions". При необходимости группировки action's можно использовать [аналогом из библиотеки react-redux](https://react-redux.js.org/api/batch)
