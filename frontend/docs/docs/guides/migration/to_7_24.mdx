## Изменения в XML API {#changes-xml-api}

- Атрибуты `first`, `first-label`, `first-icon`, `last`, `last-label`, `last-icon`, `layout`, `max-pages`,
`show-single-page` пагинации `<pagination>` упразднены.
- Атрибут `show-count` пагинации `<pagination>` теперь принимает значения `always`, `never`, `by-request`.
Для обратной совместимости в старых схемах сохранено использование `true`, `false`.
- Реализованы тип `word-wrap` для генерации кнопки переноса строк таблицы и
тип `export` для экспортирования данных таблицы.
Эти кнопки также будут генерироваться при использовании генератора `tableSettings`.

## Изменения в API автотестов {#changes-autotest-api}

- Перешли на Selenide версии 6.6.0
- Все названия методов проверок состояния элементов приведены к общему виду `shouldHave*`, `shouldBe*`
- Логика введение значений в поле ввода была изменена.
Теперь, чтобы ввести значение в поле нужно вызвать отдельно методы `click`, `clear` и `setValue`.
Метод `val` был переименован на `setValue`.
- Методы, содержащие атрибут `String regex`, были приведены к виду `*Matches(String regex)`.
