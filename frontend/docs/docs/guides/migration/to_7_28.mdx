## Изменения в сборке {#changes-build}

- Версия Java повышена с 11 на 17
- Версия Spring Boot повышена с 2.7.5 на 3.2.2
- Версии Selenide повышена c 6.17.0 на 7.3.1

## Изменения в XML API {#changes-xml-api}

- Из элемента `<list>` был удален элемент `<switch>`.
- В элементе `<list>` по умолчанию значения выводятся в столбец, а не в строку.

## Изменения Frontend {#changes-frontend}

- Обработка security-* параметров перенесана в ядро. Теперь нет необходимости производить проверку прав на отдельных компонентах.
В связи с этим удалены за ненадобностью хелперы SecurityContriller, SecurityProvider, SecurityContext.
При использовании их в кастомных компонентах, достаточно просто убрать их. Параметры, не прошедие проверку прав, будут отфильтрованы на этапе обработки метаданных страницы
